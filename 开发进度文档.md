- [x] 数据库迁移兼容性处理
- [x] 实现增删改查操作
- [x] 实现批量操作
- [x] 添加索引优化

**核心文件**：
- `app/src/main/java/com/fragmentwords/database/WordDatabase.kt` ✅

**数据库结构**：
```
words (词库表)
├── id: INTEGER (主键)
├── word: TEXT (单词，唯一)
├── phonetic: TEXT (音标)
├── translation: TEXT (翻译)
├── example: TEXT (例句)
├── difficulty: INTEGER (难度1-5)
├── part_of_speech: TEXT (词性)
└── library: TEXT (词库来源)

notebook (生词本)
├── id: INTEGER (主键)
├── word: TEXT (单词)
├── ... (其他字段同words)
└── timestamp: INTEGER (添加时间)

learning_progress (学习进度) ⭐新增
├── word: TEXT (主键，单词)
├── stage: INTEGER (复习阶段0-8)
├── next_review_time: INTEGER (下次复习时间)
├── last_review_time: INTEGER (上次复习时间)
├── review_count: INTEGER (总复习次数)
├── known_count: INTEGER (认识次数)
├── unknown_count: INTEGER (不认识次数)
└── created_time: INTEGER (首次学习时间)
```
