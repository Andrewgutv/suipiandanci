# ✅ 锁屏单词通知功能 - 使用指南

**功能类型**: 纯 JS 实现（可云打包）
**状态**: 已完成

---

## **🎯 功能说明**

这个功能使用纯 JavaScript 实现，可以在 Android 手机的通知栏和锁屏显示单词，无需原生插件，可以直接云打包。

---

## **✨ 实现的功能**

### **1. 通知栏显示单词**
- ✅ 显示单词、音标、翻译
- ✅ 显示例句
- ✅ 持续显示在通知栏
- ✅ 锁屏可见

### **2. 交互功能**
- ✅ 点击通知打开应用
- ✅ 可以开启/关闭锁屏单词
- ✅ 状态持久化保存

### **3. 定时刷新**
- ✅ 每小时自动刷新单词
- ✅ 无需手动操作
- ✅ 自动学习新单词

### **4. 权限管理**
- ✅ 自动检测通知权限
- ✅ Android 13+ 权限请求
- ✅ 引导用户开启权限

## **📊 技术实现**

### **文件结构**

```
fragment-words/
├── utils/
│   └── notification.js          # 通知工具类（新增）
├── pages/
│   └── index/
│       └── index.vue            # 首页（已更新）
└── manifest.json                # 配置文件
```

### **核心代码**

```javascript
// 使用通知工具
import notification from '@/utils/notification.js'

// 显示单词通知
notification.showWordNotification(word)

// 清除通知
notification.clearNotification()

// 开启定时刷新
notification.startAutoRefresh(60, words)  // 每60分钟

// 停止定时刷新
notification.stopAutoRefresh()
```

## **⚙️ 高级设置**

### **修改刷新间隔**

在 `pages/index/index.vue` 中找到：

```javascript
// 开启定时刷新（每60分钟）
notification.startAutoRefresh(60, this.words)
```

修改数字 `60` 为你想要的分钟数：
```javascript
// 改为每30分钟刷新
notification.startAutoRefresh(30, this.words)

// 改为每2小时刷新
notification.startAutoRefresh(120, this.words)
```

### **修改通知样式**

在 `utils/notification.js` 中的 `createPersistentNotification` 方法可以自定义通知样式。

## **🐛 常见问题**

### **Q1: 开启后通知栏看不到通知？**

**A**: 检查以下几点：
1. 确认已授予通知权限
2. 检查系统设置中的通知管理
3. 查看是否被其他应用遮挡
4. 重启应用试试

### **Q2: 锁屏后看不到单词？**

**A**:
1. 确保锁屏设置允许显示通知
2. 检查锁屏样式设置
3. 不同手机品牌可能有差异

### **Q3: 通知会自动消失吗？**

**A**: 不会。这是持续通知（Ongoing Notification），需要手动关闭或在应用中点击"关闭"按钮。

### **Q4: 定时刷新耗电吗？**

**A**: 非常省电。只是简单的定时器，不会频繁唤醒系统。

### **Q5: 可以自定义通知样式吗？**

**A**: 可以修改 `notification.js` 中的样式代码，但系统通知的样式受 Android 系统限制。
